## 1-1 과정 소개 

### 1. 클린 코드에 대해 고민하기 🧐

- 타인의 정의한 답을 의심합니다.
- 배움에 열린 태도를 가집니다.
- 직접 생각하고 또 고민합니다.
- 클린 코드가 무엇인지를 자바스크립트를 통해 학습합니다.
- 흔히 알려진 자바스크립트 코드 스타일에 대한 견해를 탐구합니다. 

<br>

### 2. JavaScript 특성 파악하기

#### 몽키 패치 (Monkey patch)
=> 런타임 중인 프로그램의 내용이 변경되는 행동을 의미한다.

#### 브랜단 아이크 ( 자바스크립트 창시자 )
=> 자바스크립트를 열흘만에 만듦
